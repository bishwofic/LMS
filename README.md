<h4>A Web Based Application</h4>
<br>
<b>
<p>Library Management System is an web application aimed for online bookkeeping. It helps librarian to maintain the database of new books and the books that are borrowed by members along with their due dates. This system completely automates all your library's activities.</p><br>
</div>
<h2>Features:</h2>
    
- User Login
- Admin Login
- User Registration
- Book Details
- Author Details
- Issue Details
- Return Details
- Category Details
- Student Information      
<br>    
    
<h2>Tools Used:</h2>
    
- HTML
- CSS
- PHP
- Javascript 
- mySQL
<br><br>

<h2>Configuration:</h2>
<h3>Database Configuration</h3>

- Open phpmyadmin
- Create Database _library_
- Import database _library.sql_
